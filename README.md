# OnePercent

A modern fitness tracking app built with React, TypeScript, and Firebase. Track your workout progress, visualize analytics, and stay motivated with real-time insights.

## Tech Stack

- React 18.3
- TypeScript
- Firebase (Auth & Firestore)
- Tailwind CSS
- Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   - Copy `.env.example` to `.env`
   - Add your Firebase configuration

3. Configure Firestore Security Rules:
   - Copy rules from `firestore.rules` to Firebase Console

4. Run development server:
   ```bash
   npm run dev
   ```

## Features

- Exercise logging with 20+ predefined exercises
- Real-time progress analytics and visualizations
- Personal records and achievement tracking
- Responsive design optimized for mobile
